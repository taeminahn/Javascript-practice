앵커 문자

메타문자

자바 스크립트 정규 표현식 객체가 갖는 가장 유용한 기능

메타문자를 이용한 주민번호 확인

추가 메타 문자

\d : 숫자
\w : 아무 단어(숫자 포함)
\s : 공백문자(탭, 띄어쓰기, 줄바꿈)
\D : 숫자가 아닌 데이터
\W : 아무 단어도 아닌 데이터(특수문자)
\S : 공백 문자가 아닌 데이터

수량 문자

(a는 예시임)
a+ : a가 적어도 1개 이상
a* : a가 0개 또는 여래개
a? : a가 0개 또는 1개
a{5} : a가 5개
a{2,5} : a가 2~5개
a{2, } : a가 2개 이상
a{ ,2} : a가 2개 이하
